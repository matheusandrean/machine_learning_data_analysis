Iniciando o portifólio de Ciência de Dados.

OBS. Aceito sugestões e projetos para estudar e por em prática.

Apresento a vocês um modelo simples, treinado para prever se o cliente irá comprar novamente ou não,
uma versão simples do modelo utilizado pelas grandes companhias de "online food service" 
que serve como base não apenas para determinarmos se um cliente a partir de suas informações irá comprar novamente, 
mas também para que então possamos saber com antecedência quais ações tomarmos para fidelizarmos aquele cliente 
e até mesmo de que forma nos prepararmos e planejarmos em relação ao "churn rate".

Neste projeto, tentei traduzir o máximo possivel e explicar da melhor forma com meu atual 
conhêcimento em inglês para que ficasse de facil compreensão para que "novatos" como eu possam captar os conceitos de um modelo 
de Machine Leaning, como analisar os dados para compreende-los, como treinar e como colocar o modelo em pratica pelo menos no notebook.

Lembrando que no "mundo real" os dados não virão em um arquivo .csv, bem estruturado, como em estudos, mas que tenhamos 
certeza de que projetos como esse servem como base para compreendermos os conceitos por de trás do "Maravilhoso Universo" dos Dados
e sua importância para as tomadas de decisões e negócios.

Como estudante e amante dos Dados, que este seja o primeiro de muitos, e que em breve eu possa estar aplicando-os em casos reais.
